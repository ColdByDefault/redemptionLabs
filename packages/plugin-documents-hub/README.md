# @coldbydefault/plugin-documents-hub

Documents Hub plugin for Redemption - Upload, organize, and manage your PDF documents securely.

## Installation

Install from GitHub Packages:

```bash
npm install @coldbydefault/plugin-documents-hub --registry=https://npm.pkg.github.com
```

## Usage

After installation, the plugin auto-registers with the Redemption app. Enable it from the Marketplace.

## Manual Import

```tsx
import { DocumentsHub } from "@coldbydefault/plugin-documents-hub";

// Use in your page
<DocumentsHub documents={documents} />;
```
